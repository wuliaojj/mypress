# Echarts笔记

##安装 cnpm install echarts --save

###1、全局引入 在main.js文件下添加下面代码 import echarts from 'echarts' Vue.prototype.$echarts = echarts

###2、使用 使用一个div来做报表的容器，一定要给宽高

<div id="myChart" style="width: 600px;height:400px;"></div>

在mounted ()钩子函数里面开始实例化我们的报表

let myChart = this.$echarts.init(document.getElementById('myChart'))

配置报表的各种参数 myChart.setOption({ title: { show: false }, tooltip: { trigger: 'axis' }, legend: { show: false }, toolbox: { show: false }, color: this.color, calculable: true, xAxis: [{ name: '产品', type: 'category', axisLine: { show: false }, axisTick: { show: false }, nameTextStyle: { color: 'rgba(255, 255, 255, 0.69)' }, axisLabel: { textStyle: { color: 'white' } }, data: ['产品1', '产品2'] }], yAxis: [{ axisLine: { show: false }, nameLocation: 'end', nameGap: 20, nameRotate: 0, axisTick: { show: false }, splitLine: { lineStyle: { color: ['rgba(230, 230, 230, 0.2)'] } }, axisLabel: { textStyle: { color: 'white', fontSize: 14 } }, name: '数量', type: 'value', nameTextStyle: { color: 'rgba(255, 255, 255, 0.69)' } }], series: [{ name: '标签1', type: 'bar', data: [2.0, 4.9], barWidth: 16, barGap: 0 }, { name: '标签2', type: 'bar', data: [2.6, 5.9], barWidth: 16, barGap: 0 }, { name: '标签3', type: 'bar', data: [2.0, 6.4], barWidth: 16, barGap: 0 }, { name: '标签4', type: 'bar', data: [4.0, 5.9], barWidth: 16, barGap: 0 }, { name: '标签5', type: 'bar', data: [5.6, 4.9], barWidth: 16, barGap: 0 }, { name: '标签6', type: 'bar', data: [2.0, 3.4], barWidth: 16, barGap: 0 }] });

完成报表的显示。

##常用配置项

title  报表标题的配置

title.show 也实现同样的功能是否像是标题 [true,flase]

title.text 也实现同样的功能标题的文字

textStyle 也实现同样的功能标题文字样式的设置 textStyle:{color:'red'}

legend 图例组件展现了不同系列的标记(symbol)，颜色和名字。可以通过点击图例控制哪些系列不显示。

grid.tooltip 数据提示框组件可以设置。

grid.tooltip.show 是否显示提示框组件，包括提示框浮层和 axisPointer。

grid.tooltip.trigger 'axis'坐标轴触发，主要在柱状图，折线图等会使用类目轴的图表中使用。

xAxis 直角坐标系 grid 中的 x 轴，一般情况下单个 grid 组件最多只能放上下两个 x 轴，多于两个 x 轴需要通过配置 offset 属性防止同个位置多个 x 轴的重叠。
